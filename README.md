# soda — クソダササムネイルジェネレーター

ワードアート・田舎の公共施設のポップ・クリップアートを彷彿とさせる、  
**クソダサ YouTube サムネイル**を一発生成するジョークツールです。

ブラウザ完結・バックエンド不要・GitHub Pages でホスティング。

## 機能

- タイトル / サブタイトルを入力するだけでクソダササムネイルが完成
- ワードアート風テキスト（グラデーション・縁取り・影・斜体・変形）
- 派手な背景（虹色グラデーション・キラキラ模様・チェック柄・水玉など）
- クリップアート風装飾（星・ハート・矢印・「!!!」など）
- 公共施設ポップ風レイアウト（吹き出し・手書き風フォント・黄地に黒縁など）
- 立ち絵アップロード → Canvas 上でドラッグ配置・ホイールでリサイズ
- 1920×1080px PNG として書き出し

## 開発

```bash
npm install
npm run dev    # 開発サーバー
npm run build  # 本番ビルド
```

## デプロイ

`main` ブランチへ push すると GitHub Actions が GitHub Pages に自動デプロイします。
